TombOfTheStupid
-
![TombOfTheStupid-Recording1](https://github.com/user-attachments/assets/08dd60f3-7ad2-497c-b404-9932a61ee725)  
is a remake of TombOfTheMask that i made just for fun and to learn something out of it.
It uses the Love2d game-framework and the Classic class module.

This version contains:
- A non blocking level generator based on prefabs called "Rooms"
- Player collision that still work on high speeds with a submove technique
- It leverages an ECS in form of Entities (here called Tiles) with corresponding Components (called Attributes)
- an collision event system where Tiles (and their attributes) can listen to Events of the Player entering tiles

I tried to keep it organized and add helping comments where I could. So feel free to tinker with it yourself.
But I am still working on it when I feel like it so don't look at it as a finished product.
